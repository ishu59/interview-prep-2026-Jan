## Handbook Structure

Each handbook follows a consistent format:

1. **Core Philosophy** -- First principles, analogies, mental models
2. **Master Templates** -- Reusable Python templates with decision matrix
3. **Pattern Classification** -- Categories and when to use each
4. **Complete Pattern Library** -- 15-20 LeetCode problems with solutions
5. **Post-Processing Reference** -- Return types and edge cases
6. **Common Pitfalls** -- Wrong vs right code examples
7. **Problem Recognition Framework** -- Decision tree for pattern matching
8. **Interview Checklist** -- Step-by-step interview strategy
9. **Quick Reference Cards** -- Condensed templates
10. **Complexity Reference** -- Time/space for all patterns

## Quick Start

1. Start with `study_roadmap.md` for a recommended order
2. Work through handbooks one pattern at a time
3. Use `algorithm_mnemonics_cheatsheet.md` for memorization
4. Use `ultimate_patterns_cheatsheet.md` as a quick reference during review
